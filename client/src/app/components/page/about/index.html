What is
fake intellect?

Fake Intellect is a blog about the gap between how smart we think we are and how smart we actually are. It is a gap that, in our experience, is mostly consistent and mostly overlooked.

We write about technology, philosophy, culture, and the ways they tangle with each other at inconvenient moments. We try to be rigorous. We mostly succeed. We update our views when we are wrong, which is often enough that we have stopped being embarrassed by it.

"The first step to not being an idiot is acknowledging that you might be one."

— Nobody said this, but they should have

The site is hosted at fakeintellect.ca and built with Angular, React, TypeScript, and a Fastify API backed by PostgreSQL. It is, fittingly, somewhat over-engineered for a blog.

If you have thoughts, corrections, or evidence that we are wrong about something, the comment sections are open. We read them. Sometimes we reply.
